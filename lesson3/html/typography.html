Typography

Heading Font:Kosugi Maru

This font is very large and not that difficult to see, so it makes it visable to anyone that would be trying to find tabs, specific parts of the page, as well as having some soft edges that could match my color patterns.

Paragraph Font: Sans Serif

I went with this font because it was one that paired with the heading font, as well as having open clear characters.
